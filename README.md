# 🎮 Tic Tac ToEs (Chaotic Edition)

Welcome to **Tic Tac ToEs**, a wildly energetic, slightly unhinged twist on the classic Tic Tac Toe game — built with Python and Tkinter.

This isn’t your average grid game… expect random nicknames, dramatic popups, and *a LOT of shouting* 😄

---

## ✨ Features

* 🎲 **Random Nicknames**
  Your real name? Irrelevant. The game assigns you something *way better*.

* 🗣️ **Interactive Dialogs**
  The game talks to you. Loudly. And sometimes aggressively.

* 🏆 **Best-of Series Gameplay**
  Players decide how many rounds to play — first to the winning score becomes the **CHAMPION**.

* 🧠 **Win Detection System**
  Detects wins, losses, and draws automatically.

* 🔄 **Reset & New Game Options**

  * Reset the board anytime
  * Start a completely new chaotic experience

* 💰 **Totally Real Prize System**
  Winner earns:

  > $100,000,000,000,000,000,000
  > *(definitely not a scam 🙃)*
